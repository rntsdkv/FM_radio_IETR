# ИЭТР: FM-передатчик на трёх транзисторах

Интерактивное электронное техническое руководство (ИЭТР) 4-го класса на
маломощный FM-передатчик с микрофоном. Три транзистора 2N3904:
микрофонный усилитель, предусилитель и ВЧ-генератор с частотной модуляцией.

Сайт содержит описание принципа работы, интерактивную 3D-модель платы,
принципиальную схему, топологию, технические характеристики и полный
перечень элементов.

## Стек

- React 19
- Vite 6
- Three.js (интерактивный просмотр `.glb`-модели)

## Быстрый старт

```bash
npm install
npm run dev
```

Сборка продакшена:

```bash
npm run build
npm run preview
```

## Публикация на GitHub Pages

1. В `vite.config.js` при необходимости укажите `base: "/<имя-репозитория>/"`.
2. Выполните:

```bash
npm run deploy
```

## Структура

```
public/
  models/board.glb        — 3D-модель платы (glTF 2.0)
  images/                 — схема, топология, рендер
src/
  data.js                 — характеристики и перечень элементов (правьте здесь)
  App.jsx                 — секции страницы
  components/BoardViewer.jsx — 3D-просмотрщик на Three.js
  styles/global.css       — оформление (тёмная неон-тема)
```

Чтобы изменить характеристики или перечень элементов — редактируйте
`src/data.js`, всё остальное подхватится автоматически.
